⬇️ **配置流程：**

- 第一次使用需要将输入光标放到鼠须管快捷短语配置文件的新行，然后保存
- 在 Macro 中将文件路径改为自己的鼠须管快捷短语配置文件路径
- 系统语言为中文的话，可能要选择一下 Macro 中的输入法名字
- 通过手动运行或者 LaunchBar 运行
